# Docentes
Frontend de Docentes

Requisitos del sistema
Antes de ejecutar este proyecto Angular, asegúrate de tener instalado lo siguiente en tu computadora:

 -node
 -npm

Verifica si tienes esto ejecutando desde la consola:

node -v
npm -v

EN CASO DE NO TENER INSTALADO:

- Descarga recomendada: https://nodejs.org
- Versión sugerida: Node.js 18.x o superior
- npm se instala automáticamente con Node.js

Para descargar Angular CLI:

EJECUTAR DESDE LA CONSOLA
npm install -g @angular/cli

Para verificar que se instaló correctamente:

ng version

Una vez clonado el repositorio, ejecutar:

npm install

Este comando instalará topdas las dependencias necesarias definidas en package.json.

Servidor de desarrollo
Para iniciar un servidor de desarrollo local, ejecutar en la terminal:

ng serve

Una vez que el servidor esté en funcionamiento tanto este como el de backend, abre tu navegador y navega a http://localhost:4200/login. De aqui en adelante se darán las credenciales y explicaciones durante la defensa.
